Create Navigational Links within a Website

There are several techniques to navigate within a website. The semantic tag for navigation is the <nav< tag. You can navigate through your website by using relaltive addressing. For our examples we will use the unordered list tag (<ul<)

  • Page to page within the same folder
  • Page to page between folders
  • Staying on the current page
  • Going to a specific place on the current page (on page navigation)

Navigational menus are placed in your <header< semantic area. They may also be placed in your <footer< area as a site map. The nav tag, like other semantic tags, should have a <hx< tag, HOWEVER this header is optional. If you omit the header tag you will recieve a document outliner error: Untitled Section. Until you are shown how to avoid this error message, it will be acceptable on your document outline.
